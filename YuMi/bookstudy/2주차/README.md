> 슬라이스식으로 원소에 접근 하기
   
 - s[i:j] : s[i]부터 s[j-1]까지 나열
 
|구분|설명|
|----|-----|
|s[:]| 리스트s의 원소를 모두 출력|
|s[:n]| 리스트s의 원소 중에 맨 앞부터 n개까지 출력|
|s[i:]| 리스트s의 원소 중 s[i]부터 맨 끝까지 출력|
|s[-n:]| 리스트s의 원소 중 -n부터 맨 끝까지 출력|
|s[::k]|리스트s의 원소 중 맨 앞부터 k개씩 건너뛰며 출력|
|s[::-1]|리스트s의 원소 중 맨 끝부터 전부 출력|
